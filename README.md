# Flower Order App

## Опис
Навчальний Android-застосунок, створений у середовищі **Android Studio** в рамках лабораторної роботи №1.  
Застосунок реалізує простий інтерфейс замовлення квітів із використанням елементів керування.

## Функціональність
- Введення назви квітів
- Вибір кольору (RadioButton)
- Вибір діапазону цін (RadioButton)
- Перевірка заповнення всіх полів
- Виведення інформації про замовлення
- Повідомлення (Toast) у разі неповного введення даних

## Використані технології
- Android Studio
- Java
- XML (Layout)
- Android SDK

## Запуск проєкту
1. Відкрити проєкт в Android Studio
2. Дочекатися завершення Gradle Sync
3. Запустити застосунок на емуляторі або фізичному пристрої

## Призначення
Проєкт створено з навчальною метою для ознайомлення з базовими елементами інтерфейсу Android та обробкою подій користувача.
